Check if a given string has all symbols in upper case. (Empty string should return True)

Input: String.

Output: Boolean.

Example:

{% if interpreter.slug == "js-node" %} 
isAllUpper('ALL UPPER') == true
isAllUpper('all lower') == false
isAllUpper('mixed UPPER and lower') == false
isAllUpper('') == true
{% else %} 
is_all_upper('ALL UPPER') == True
is_all_upper('all lower') == False
is_all_upper('mixed UPPER and lower') == False
is_all_upper('') == True
{% endif %}

Precondition: a-z, A-Z, 1-9 and space
